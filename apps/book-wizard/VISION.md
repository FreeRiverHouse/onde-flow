# Book Wizard — Vision

## Cos'è
Book Wizard è il modulo di Onde-Flow per la gestione editoriale AI-driven.
Gestisce pipeline completa: importazione EPUB → editing capitoli → traduzione → export PDF/EPUB.

## Progetto attuale: Piccoli Sutra per il Risveglio
Raccolta di testi spirituali in fase di traduzione e pubblicazione digitale.

## Obiettivo a lungo termine
- Editor capitoli con AI assistant (correzione, stile, coerenza)
- Traduzione automatica multi-lingua con revisione umana
- Export professionale: EPUB, PDF, stampa on-demand
- Gestione catalogo completo Onde Casa Editrice

## Stack tecnico
- Node.js pipeline (EPUB parsing, PDF generation)
- Claude (editing, traduzione, revisione stile)
- Supabase (catalogo libri, versioni)
- Next.js dashboard (controllo editoriale)
